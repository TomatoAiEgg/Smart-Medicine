# 中间件连接信息

本文档记录开发环境中 PostgreSQL、Redis、RocketMQ 的连接方式和用途说明。

敏感密码不写入本文档明文；本地运行时通过环境变量或交互输入提供。

## 当前开发环境连接

当前项目中间件运行在老笔记本 Ubuntu，通过 Tailscale 访问。

| 组件 | 地址 | 端口 | 说明 |
| --- | --- | --- | --- |
| PostgreSQL | `100.115.165.51` | `15432` | 项目专用库，容器内库名为 `zhyf_saas` |
| Redis | `100.115.165.51` | `16379` | 项目专用 Redis，启用密码 |
| RocketMQ NameServer | `100.115.165.51` | `9876` | 项目专用 RocketMQ NameServer |
| RocketMQ Broker | `100.115.165.51` | `10909`、`10911`、`10912` | 项目专用 RocketMQ Broker |
| RocketMQ Dashboard | `100.115.165.51` | `18181` | RocketMQ 控制台 |
| Grafana | `100.115.165.51` | `13000` | 开发联调观测控制台 |
| Loki | `100.115.165.51` | `13100` | Docker 容器日志查询 |
| Prometheus | `100.115.165.51` | `19090` | 指标采集与查询 |

## 注意事项

- 原云服务器上的 `zhyf-*` 容器已删除，本项目中间件已迁移到老笔记本 Ubuntu。
- 本项目默认通过 Tailscale 地址 `100.115.165.51` 访问中间件。
- 本地 Redis / RocketMQ / PostgreSQL 默认不启动。
- 文档只记录端口和用途，不记录明文密码。
- 后端配置应通过环境变量、启动参数或本地未提交配置文件注入。

## 服务配置键建议

| 配置项 | 示例 |
| --- | --- |
| `spring.datasource.url` | `jdbc:postgresql://100.115.165.51:15432/zhyf_saas` |
| `spring.datasource.username` | 从环境变量读取 |
| `spring.datasource.password` | 从环境变量读取 |
| `spring.data.redis.host` | `100.115.165.51` |
| `spring.data.redis.port` | `16379` |
| `spring.data.redis.password` | 从环境变量读取 |
| `rocketmq.name-server` | `100.115.165.51:9876` |

## 验证项

- PostgreSQL：能连接项目库并执行简单查询。
- Redis：能执行 `PING` 并返回正常响应。
- RocketMQ：Dashboard 显示 NameServer、Broker、Topic、ConsumerGroup 状态正常。
- 观测：Grafana `/api/health`、Loki `/ready`、Prometheus `/-/ready`、Alloy `/-/ready`、cAdvisor `/healthz` 正常。
- 端口：本项目只使用 `15432`、`16379`、`9876`、`10909`、`10911`、`10912`、`18181`、`13000`、`13100`、`19090`、`12345`、`19100`、`18084` 等已规划端口。
