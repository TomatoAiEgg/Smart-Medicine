# 观测中间件

本文档记录老笔记本 Ubuntu 上为本项目准备的轻量级观测能力。当前机器内存有限，不引入 ELK 这类重组件，优先使用 Grafana、Loki、Alloy、Prometheus、node-exporter 和 cAdvisor。

## 部署位置

| 项 | 值 |
| --- | --- |
| 主机 | 老笔记本 Ubuntu |
| Tailscale 地址 | `100.115.165.51` |
| SSH 用户 | `tomatoegg` |
| 部署目录 | `/home/tomatoegg/zhyf-observability` |
| Compose 网络 | `zhyf-observability` |

SSH 私钥只保存在本地，不写入仓库文档。Grafana 管理员密码保存在远端 `.env`，文档不记录明文密码。

## 组件与端口

| 组件 | 容器名 | 镜像 | 地址 |
| --- | --- | --- | --- |
| Grafana | `zhyf-grafana` | `grafana/grafana:11.6.8` | `http://100.115.165.51:13000` |
| Loki | `zhyf-loki` | `grafana/loki:3.6.3` | `http://100.115.165.51:13100` |
| Alloy | `zhyf-alloy` | `grafana/alloy:v1.11.3` | `http://100.115.165.51:12345` |
| Prometheus | `zhyf-prometheus` | `prom/prometheus:v3.8.1` | `http://100.115.165.51:19090` |

Grafana 已自动配置 Loki 和 Prometheus 数据源。Loki 通过 Alloy 采集 Docker 容器日志，当前可查询的基础标签包括 `container`、`compose_project`、`service_name`。

## 运维命令

在老笔记本 Ubuntu 上执行：

```text
cd /home/tomatoegg/zhyf-observability
sudo docker compose ps
sudo docker compose up -d
sudo docker compose logs --tail 100 zhyf-grafana
sudo docker compose logs --tail 100 zhyf-loki
sudo docker compose logs --tail 100 zhyf-alloy
```

重置 Grafana 管理员密码时，先更新远端 `.env` 中的 `ZHYF_GRAFANA_ADMIN_PASSWORD`，再执行：

```text
cd /home/tomatoegg/zhyf-observability
sudo docker exec zhyf-grafana grafana cli admin reset-admin-password "$ZHYF_GRAFANA_ADMIN_PASSWORD"
sudo docker compose up -d zhyf-grafana
```

## 使用边界

- 当前观测环境用于开发联调，不作为生产级高可用监控。
- Prometheus 数据保留时间为 7 天。
- Loki 用于容器日志排查，不长期保存大量业务审计数据。
- 老笔记本内存有限，新增组件前先检查 `free -h` 和 `docker stats`。
- 不在本机额外启动日志、监控或数据库组件，避免和老笔记本环境重复消耗资源。
