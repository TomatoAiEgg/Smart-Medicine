## 第一版业务联调清单

| 顺序 | 联调链路 | 验收点 |
| --- | --- | --- |
| 1 | 机构推单 -> 订单/处方/明细落库 | 幂等、签名、IP 白名单、状态初始值正确 |
| 2 | 订单创建 -> Outbox -> RocketMQ | 本地事务一致，事件可发布，消费幂等 |
| 3 | 后台审核/调剂/复核 | 状态机合法推进，状态日志完整 |
| 4 | PDA 模拟扫码绑定 | 重复扫码、并发扫码、超时重试可控 |
| 5 | MES 模拟开始/完成煎煮 | 乱序、重复上报可识别，设备占用可释放 |
| 6 | 打包出库 -> 物流轨迹 -> 签收 | 物流状态和订单状态联动正确 |
| 7 | 签收/取消/异常 -> 机构回调 | 回调签名、失败重试、死信、人工重放完整 |
| 8 | 社康/医院/地址补录适配 | 外围协议兼容，差异不侵入订单主流程 |
